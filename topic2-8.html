String Operations & Nesting Quotes

Key Concepts

  • Using single (') vs. double (") quotes
  • String concatenation using +
  • Template literals using backticks `
  • Escape characters like \' and \"
  • eval() (⚠️ avoid using in real projects)

Example Code

let singleQuote = 'It\'s a beautiful day!';
let doubleQuote = "She said, \"Hello!\"";
let templateLiteral = `Hello, ${singleQuote}`;

console.log(singleQuote);       // It's a beautiful day!
console.log(doubleQuote);       // She said, "Hello!"
console.log(templateLiteral);   // Hello, It's a beautiful day!

console.log(eval("2 + 2"));     // 4

⚠️ Warning: eval() executes code from a string. Avoid it due to security risks.

Practice Questions

  1. let text = "He said, \"JavaScript is awesome!\"";
console.log(text.length);  // 29
          
  2. function getFullName(firstName, lastName) {
    return `${firstName} ${lastName}`;
}
console.log(getFullName("Aman", "Raj")); // "Aman Raj"
          
  3. console.log(eval("2 + 2 * 2")); // 6
// Explanation: Evaluates as 2 + (2 * 2)
