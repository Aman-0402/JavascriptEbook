2.3 Perform Type Conversions

JavaScript allows converting one type of data to another either automatically or manually.

2.3.1 Implicit Type Conversion (Coercion)

JavaScript automatically converts values from one type to another when required.

console.log("5" + 2); // "52" (string)
console.log("5" - 2); // 3 (number)
console.log(true + 1); // 2

2.3.2 Explicit Type Conversion

Use built-in functions like Number(), String(), Boolean(), and parseInt() to manually convert data types.

let numStr = "123";
let num = Number(numStr); // Converts string to number
console.log(num + 1); // 124

let boolVal = Boolean(0); // false
console.log(boolVal);

Practice Questions

  1. Answer:

    let year = "2025";
let numYear = Number(year);
console.log(numYear + 5); // Output: 2030

  2. Answer:

    console.log(Boolean(0)); // Output: false
console.log(Boolean(1)); // Output: true

  3. Answer:

    let boolValue = true;
let strBool = String(boolValue);
console.log(strBool); // Output: "true"
