3.3 Loops

Explanation

Loops help us repeat actions multiple times.

  1. for Loop
    Runs a block a fixed number of times. 
    for (let i = 1; i <= 5; i++) {
    console.log("Iteration " + i);
}
  2. for...in Loop
    Used for objects to loop through keys. 
    let person = { name: "Aman", age: 25, city: "Delhi" };
for (let key in person) {
    console.log(key + ": " + person[key]);
}
  3. while Loop
    Runs while a condition is true. 
    let count = 1;
while (count <= 5) {
    console.log(count);
    count++;
}
  4. do...while Loop
    Runs at least once, then repeats while true. 
    let num = 1;
do {
    console.log("Value: " + num);
    num++;
} while (num <= 3);
  5. break and continue
    • break → Stops the loop early.
    • continue → Skips the current iteration.
    for (let i = 1; i <= 5; i++) {
    if (i == 3) break;  // Stops at 3
    console.log(i);
}

for (let i = 1; i <= 5; i++) {
    if (i == 3) continue;  // Skips 3
    console.log(i);
}
