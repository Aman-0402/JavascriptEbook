3.3 Loops in JavaScript

Loops help us repeat actions multiple times.

1. for Loop

Runs a block a fixed number of times.

for (let i = 1; i <= 5; i++) {
    console.log("Iteration " + i);
}
Iteration 1
Iteration 2
Iteration 3

Iteration 5

2. for...in Loop

Used for objects to loop through keys.

let person = { name: "Aman", age: 25, city: "Delhi" };
for (let key in person) {
    console.log(key + ": " + person[key]);
}
name: Aman
age: 25
city: Delhi

3. while Loop

Runs while a condition is true.

let count = 1;
while (count <= 5) {
    console.log(count);
    count++;
}

2
3
4

4. do...while Loop

Runs at least once, then repeats while condition is true.

let num = 1;
do {
    console.log("Value: " + num);
    num++;
} while (num <= 3);
Value: 1
Value: 2
Value: 3

5. break and continue

break stops the loop early. continue skips the current iteration.

for (let i = 1; i <= 5; i++) {
    if (i == 3) break;
    console.log(i);
}
