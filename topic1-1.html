1.1 Assignment and Arithmetic Operators

JavaScript provides various operators for performing arithmetic and assignment operations.

1.1.1 Assignment Operators

Assignment operators are used to assign values to variables.

Operator Example Equivalent To Description
= x = 10 - Assigns 10 to x
+= x += 5 x = x + 5 Adds and assigns
-= x -= 3 x = x - 3 Subtracts and assigns
*= x *= 2 x = x * 2 Multiplies and assigns
/= x /= 4 x = x / 4 Divides and assigns
%= x %= 3 x = x % 3 Finds remainder and assigns

1.1.2 Arithmetic Operators

Arithmetic operators are used for mathematical calculations.

Operator Example Result
+ 5 + 3 8
- 10 - 4 6
* 6 * 3 18
/ 12 / 4 3
% 10 % 3 1 (Remainder)

1.1.3 Increment and Decrement Operators

  • ++ (Increment) increases the value by 1.
  • -- (Decrement) decreases the value by 1.
let x = 5;
console.log(x++); // Outputs 5 (Post-increment: returns the value before incrementing)
console.log(x);   // Outputs 6

let y = 5;
console.log(++y); // Outputs 6 (Pre-increment: increments first, then returns value)
