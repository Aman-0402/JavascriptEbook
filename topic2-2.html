2.2 Identify and Use Different Data Types

JavaScript supports multiple data types categorized into primitive and non-primitive (reference) types.

2.2.1 Primitive Data Types

Data TypeExampleDescription
String"Hello"Sequence of characters
Number42, 3.14Integer or decimal
Booleantrue, falseLogical values
NullnullIntentional empty value
Undefinedlet x;Declared but not assigned
SymbolSymbol("id")Unique and immutable
BigInt9007199254740991nFor large integers
let str = "JavaScript";
let num = 100;
let isAvailable = true;
let emptyValue = null;
let notAssigned;
console.log(typeof str, typeof num, typeof isAvailable, typeof emptyValue, typeof notAssigned);

2.2.2 Non-Primitive (Reference) Data Types

Data TypeExampleDescription
Object{name: "Aman", age: 25}Key-value pairs
Array[1, 2, 3]List of values
Functionfunction() { return "Hi"; }Block of code
let person = { name: "Aman", age: 25 };
let colors = ["red", "blue", "green"];
let greet = function() { return "Hello!"; };
console.log(typeof person, typeof colors, typeof greet);
