5.1 Complete and Debug Code That Retrieves Form Input and Sets Form Field Values

‚úÖ 1. Retrieving Form Input Values

To get the value entered in a form field, use the value property.

üîπ Example: Get input value on button click

<label for="name">Enter your name:</label>
<input type="text" id="name" placeholder="Type here">
<button onclick="getInputValue()">Submit</button>
<p id="output"></p>

<script>
  function getInputValue() {
    let nameValue = document.getElementById("name").value.trim();
    if (nameValue === "") {
      alert("Please enter your name!");
      return;
    }
    document.getElementById("output").textContent = "Hello, " + nameValue;
  }
</script>

üîç Explanation:

  • document.getElementById("name").value ‚Üí Retrieves the user input.
  • textContent ‚Üí Displays the input in a <p> tag.

‚úÖ Explanation of the Code

This HTML and JavaScript code allows a user to enter their name in an input field and displays a greeting message when they click the "Submit" button.

üéØ Step-by-Step Breakdown:

  1. HTML Structure:
    • <label> and <input> allow user input.
    • <button> triggers the JS function.
    • <p> displays the message.
  2. JavaScript Function (getInputValue()):
    • Retrieves input using .value.trim().
    • Shows greeting inside the paragraph.

üöÄ Enhancements & Best Practices

‚úÖ 1. Use addEventListener Instead of onclick

‚úÖ Why? Keeps JavaScript separate from HTML, making code cleaner and reusable.

üîç Explanation:

  • document.getElementById("email").value = "example@gmail.com"; ‚Üí Sets the input field with a default email.

‚úÖ Explanation of the Code:

  • This HTML and JavaScript code allows a user to set a predefined email in the input field when they click the button.

üéØ Step-by-Step Breakdown:

  1. HTML Structure:
    • A <label> and <input> for email input.
    • A <button> that triggers setEmail().
  2. JavaScript Function (setEmail()):
    • Uses getElementById to access the input field.
    • Sets value to "example@gmail.com".

üöÄ Enhancements & Best Practices

  • ‚úÖ Use addEventListener Instead of onclick
