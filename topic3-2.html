3.2 Decision Statements

Explanation

  1. Single Alternative (if)
    Executes a block if the condition is true.
    let num = 10;
if (num > 5) {
    console.log("Number is greater than 5");
}
  2. Dual Alternative (if-else)
    Executes one block if true, another if false.
    let marks = 75;
if (marks >= 50) {
    console.log("Pass");
} else {
    console.log("Fail");
}
  3. Multiple Alternatives (switch)
    Used when multiple cases exist.
    let day = "Monday";
switch (day) {
    case "Monday":
        console.log("Start of the week!");
        break;
    case "Friday":
        console.log("Weekend is near!");
        break;
    default:
        console.log("Just another day.");
}
  4. Nested if
    if inside another if to check multiple conditions.
    let score = 85;
if (score >= 50) {
    if (score >= 80) {
        console.log("Excellent!");
    } else {
        console.log("Good job!");
    }
} else {
    console.log("Try again!");
}

Practice Questions

  1. Write a program that takes a temperature as input and:
    • Prints "Hot" if temp is above 30°C
    • Prints "Warm" if temp is between 20-30°C
    • Prints "Cold" otherwise
  2. What will be the output? 
    let a = 5;
let b = 10;
if (a > b) console.log("A is greater");
else if (a == b) console.log("Both are equal");
else console.log("B is greater");
  3. Write a switch-case program that takes an input month (1-12) and prints the season:
    • Summer: March - June
    • Monsoon: July - September
    • Winter: October - February

Answers

  1. Temperature Checker 
    let temp = 25;
if (temp > 30) {
    console.log("Hot");
} else if (temp >= 20 && temp <= 30) {
    console.log("Warm");
} else {
    console.log("Cold");
}
    Output (if temp = 25): Warm
  2. Predict the Output 
    let a = 5;
let b = 10;
if (a > b) console.log("A is greater");
else if (a == b) console.log("Both are equal");
else console.log("B is greater");
    Output: B is greater
  3. Switch-Case for Seasons 
    let month = 5; // May
switch (true) {
    case (month >= 3 && month <= 6):
        console.log("Summer");
        break;
    case (month >= 7 && month <= 9):
        console.log("Monsoon");
        break;
    case (month >= 10 || month <= 2):
        console.log("Winter");
        break;
    default:
        console.log("Invalid month");
}
    Output (if month = 5): Summer
