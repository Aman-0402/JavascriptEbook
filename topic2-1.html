2.1 Declare and Initialize Variables

In JavaScript, variables store data values. They can be declared using var, let, or const.

2.1.1 Using var, let, and const

  • var: Function-scoped, can be re-declared and updated.
  • let: Block-scoped, can be updated but not re-declared.
  • const: Block-scoped, cannot be updated or re-declared.
var name = "Aman";    // Function-scoped
let age = 25;         // Block-scoped
const PI = 3.14159;   // Cannot be reassigned

2.1.2 Variable Initialization

  • A variable can be initialized when declared.
  • If declared but not initialized, it holds undefined.
let city;
console.log(city); // Output: undefined
city = "Delhi";
console.log(city); // Output: Delhi

Practice Questions

  1. Answer:

    var x = 10;
let y = 20;
const z = 30;
console.log(x, y, z); // Output: 10 20 30

  2. Answer:

    console.log(x); 
var x = 5;

// Output: undefined
// Reason: Due to hoisting, the declaration of x is moved to the top but not the assignment.

  3. Answer:

    const GRAVITY = 9.8;
GRAVITY = 10; // Error: Assignment to constant variable.

// JavaScript throws a TypeError because constants cannot be reassigned.
