1.3 Evaluate the Use of Internal and External Scripts

JavaScript can be included in HTML in two ways:

1.3.1 Internal Script

Placed inside <script> tag in HTML:

<script>
    function greet() {
        alert("Hello, World!");
    }
</script>

1.3.2 External Script

Stored in a .js file and linked in HTML:

<script src="script.js"></script>

Advantages of External Scripts:

  • Improves performance (cached by browser).
  • Enhances code reusability.
  • Keeps HTML cleaner.

What happens when scripts are used at multiple levels?

  • Internal script overrides external script if placed after <script src="script.js">.
  • If multiple external scripts modify the same variable, the last loaded script takes precedence.

Practice Questions

  1. Answer:

    <!DOCTYPE html>
<html lang="en">
<head>
    <title>Internal Script</title>
</head>
<body>
    <script>
        alert("Hello from Internal Script!");
    </script>
</body>
</html>

  2. Answer:

    // script.js
function showMessage() {
    alert("Hello from External Script!");
}

// Link this in an HTML file:
<script src="script.js"></script>

  3. Answer:

    <!DOCTYPE html>
<html lang="en">
<head>
    <script src="script.js"></script>
    <script>
        function showMessage() {
            alert("This will override the external script!");
        }
    </script>
</head>
<body>
    <button onclick="showMessage()">Click Me</button>
</body>
</html>

// What happens?
// The internal script will override the external script if defined later in the HTML file.
