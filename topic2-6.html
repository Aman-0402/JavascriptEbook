2.6 Differentiate Between Function Scope and Block Scope

2.6.1 Function Scope (var)

function example() {
    var x = 30;
    if (true) {
        var y = 20;
    }
    console.log(y); // Works because var is function-scoped
}
example();

2.6.2 Block Scope (let, const)

function example() {
    if (true) {
        let x = 10;
        const y = 20;
    }
    console.log(x); // Error: x is not defined
}

Practice Questions

  1. Answer:

    Variables declared with var have function scope, so they are accessible throughout the function, including outside an if block.

    Variables declared with let or const are limited to the block ({}) where they are defined.

    if (true) {
    var a = 10;
    let b = 20;
}
console.log(a); // Output: 10
console.log(b); // Error: b is not defined

  2. Answer:

    function testScope() {
    var x = 10; // Function scope
    if (true) {
        let y = 20; // Block scope
        console.log(x); // Accessible
        console.log(y); // Accessible
    }
    console.log(x); // Accessible
    console.log(y); // Error: y is not defined
}
testScope();

  3. Code:

    function test() {
    let a = 10;
    if (true) {
        let a = 20;
        console.log(a);
    }
    console.log(a);
}
test();

    Output:

    20
10

    Explanation:

    The inner let a = 20 is block-scoped and separate from the outer let a = 10.
