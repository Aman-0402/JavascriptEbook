Input validation ensures users provide correct and expected values before submission.

üîπ 1. Case Sensitivity & String Comparisons

Ensure input matches specific text regardless of case.





    <input type="text" id="username" placeholder="Enter username">
    <button onclick="validateUsername()">Submit</button>
    <p id="output"></p>
    
    <script>
      function validateUsername() {
        let input = document.getElementById("username").value.trim();
        let validUser = "admin"; // Expected username (lowercase)
        if (input.toLowerCase() === validUser) {
          document.getElementById("output").textContent = "Valid username!";
        } else {
          document.getElementById("output").textContent = "Invalid username!";
        }
      }
    </script>

üîç Explanation:

  • .trim() removes extra spaces.
  • .toLowerCase() ensures case-insensitive comparison.

üîπ 2. Number Validation (isNaN())

Check if input is a valid number.





    <input type="text" id="age" placeholder="Enter your age">
    <button onclick="validateAge()">Submit</button>
    <p id="message"></p>
    
    <script>
      function validateAge() {
        let age = document.getElementById("age").value;
        if (age === "") {
          document.getElementById("message").textContent = "Age cannot be blank!";
        } else if (isNaN(age)) {
          document.getElementById("message").textContent = "Please enter a valid number!";
        } else {
          document.getElementById("message").textContent = "Valid age!";
        }
      }
    </script>

üîç Explanation:

  • isNaN(value) checks if value is not a number.
  • The first condition ensures the input is not empty.

üîπ 3. Not Blank Validation

Ensure input is not empty.





    <input type="email" id="email" placeholder="Enter email">
    <button onclick="validateEmail()">Submit</button>
    <p id="emailMsg"></p>
    
    <script>
      function validateEmail() {
        let email = document.getElementById("email").value.trim();
        if (email === "") {
          document.getElementById("emailMsg").textContent = "Email cannot be blank!";
        } else {
          document.getElementById("emailMsg").textContent = "Valid email!";
        }
      }
    </script>

üîç Explanation:

  • .trim() prevents spaces from being counted.
