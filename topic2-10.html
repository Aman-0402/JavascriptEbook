Objects & Date Object

Key Concepts

  • Accessing object keys and values with Object.keys() and Object.values()
  • Working with Date objects
  • Formatting dates using toLocaleDateString()
  • Modifying date values (adding/subtracting days)

Example Code

let today = new Date();
console.log(today.toLocaleDateString()); // "MM/DD/YYYY" or "DD/MM/YYYY" based on locale

let specificDate = new Date(2024, 11, 25); // Dec 25, 2024
console.log(specificDate);

specificDate.setDate(specificDate.getDate() + 5); // Adds 5 days
console.log(specificDate);

Practice Questions

  1. let car = { brand: "Toyota", model: "Camry", year: 2022 };
car.year = 2023;
console.log(car); // { brand: "Toyota", model: "Camry", year: 2023 }
          
  2. let person = { name: "John", age: 25 };
console.log(Object.keys(person));   // ["name", "age"]
console.log(Object.values(person)); // ["John", 25]
          
  3. let myBirthday = new Date(2000, 5, 15); // June 15, 2000
console.log(myBirthday.toLocaleDateString()); // Output depends on locale
