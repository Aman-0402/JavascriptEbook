Arrays & Multi-Dimensional Arrays

Key Concepts

  • Array sorting using sort() and reverse()
  • Array searching with indexOf() and includes()
  • Multi-dimensional arrays
  • Modifying arrays: push(), pop(), shift(), unshift()

Example Code

let fruits = ["Apple", "Banana", "Cherry"];
fruits.push("Dragonfruit");
fruits.pop();
fruits.shift();
fruits.unshift("Mango");

console.log(fruits); // [ 'Mango', 'Banana' ]

let numbers = [5, 2, 8, 1, 9];
numbers.sort((a, b) => a - b);  // Sorts in ascending order
console.log(numbers); // [1, 2, 5, 8, 9]
console.log(numbers.indexOf(8)); // 3
console.log(numbers.includes(10)); // false

let matrix = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];
console.log(matrix[1][2]); // 6
