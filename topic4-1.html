4.1 Identify and Construct the Document Object Model (DOM) Tree

What is the DOM?

The Document Object Model (DOM) represents an HTML document as a tree where each element (like <html>, <body>, <p>) is a node. JavaScript can be used to access, modify, and manipulate the DOM.

DOM Tree Structure

When a webpage loads, the browser converts the HTML into a structured DOM tree.

Example HTML Page

<!DOCTYPE html>
<html>
<head>
    <title>DOM Example</title>
</head>
<body>
    <h1 id="title">Hello, DOM!</h1>
    <p class="info">This is a paragraph.</p>
    <button onclick="changeText()">Click Me</button>
</body>
</html>

Corresponding DOM Tree

window
 ├── document
      ├── html
          ├── head
          │    └── title
          └── body
               ├── h1 (id="title")
               ├── p (class="info")
               └── button (onclick="changeText()")

Important DOM Objects

  • window → Represents the browser window (highest-level object).
  • document → Represents the webpage.
  • document.body → Represents the <body> tag.
  • document.head → Represents the <head> tag.
  • Other HTML elements → Access using methods like:
    • document.getElementById()
    • document.getElementsByClassName()

JavaScript Example: Accessing the DOM

// Access the title element
let heading = document.getElementById("title");
console.log(heading.innerText); // Output: "Hello, DOM!"


// Change text using JavaScript
function changeText() {
    heading.innerText = "DOM Manipulated!";
}

Practice Questions

  1. Answer:

    The DOM tree is a structured representation of an HTML document. It is important because JavaScript uses it to access and manipulate webpage content dynamically, enabling interactivity and real-time updates. 

  2. <div>
  <h2>Welcome</h2>
  <p>This is a paragraph.</p>
</div>

    Answer:

    Parent: <div>
    Children: <h2> and <p>
    Siblings: <h2> and <p> (they are siblings of each other)

  4. console.log(document.body.innerHTML);
console.log(document.head.innerText);

    Answer:

    document.body.innerHTML prints all inner HTML inside the <body>.
    document.head.innerText prints the visible text content inside the <head>, such as the title.
