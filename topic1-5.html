1.5 Complete and Debug Code That Interacts with the BOM

The Browser Object Model (BOM) allows interaction with the browser.

1.5.1 Displaying Dialogs

  • Alert: alert("Hello!")
  • Confirm: 
    let confirmAction = confirm("Are you sure?");
console.log(confirmAction);
  • Prompt: 
    let name = prompt("Enter your name:");
alert("Welcome, " + name);

1.5.2 Determining Screen Size

console.log("Screen Width:", window.screen.width);
console.log("Screen Height:", window.screen.height);
