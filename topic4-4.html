üîπ 1. getElementById()

‚úÖ Use: Find a single HTML element by its id.

‚úÖ Example:

<p id="demo">Hello</p>
<button onclick="changeText()">Click Me</button>

<script>
  function changeText() {
    const element = document.getElementById("demo");
    element.textContent = "Text changed using getElementById!";
  }
</script>

üîç What Happens Here?

üìå Initial State:
  • A paragraph (<p>) with id="demo" displays: Hello
üéØ When the Button is Clicked:
  • The function changeText() is triggered.
  • const element = document.getElementById("demo"); grabs the <p> element.
  • element.textContent = "Text changed using getElementById!"; updates its text.

üß† What's textContent?

  • element.textContent sets or gets the text inside an element.
  • Unlike innerHTML, it treats the content as plain text (not HTML).

Action Output
Page Load Hello
Button Click Text changed using getElementById!

üîπ 2. getElementsByTagName()

‚úÖ Use: Find all elements with a specific HTML tag name. Returns a collection (like an array).

‚úÖ Example:

      <p>First paragraph</p>
      <p>Second paragraph</p>
      <button onclick="colorParagraphs()">Change Color</button>
      
      <script>
        function colorParagraphs() {
          const paragraphs = document.getElementsByTagName("p");
          for (let i = 0; i < paragraphs.length; i++) {
            paragraphs[i].style.color = "blue";
          }
        }
      </script>

üîç How It Works

  • ‚úÖ You have two <p> elements on the page.
  • ‚úÖ A <button> triggers the colorParagraphs() function when clicked.
  • ‚úÖ Inside the function: 
    const paragraphs = document.getElementsByTagName("p");
    • This grabs all <p> tags on the page and returns an HTMLCollection (like an array).
  • ‚úÖ The for loop: 
          for (let i = 0; i < paragraphs.length; i++) {
        paragraphs[i].style.color = "blue";
      }
    • Iterates through each paragraph.
    • Sets their style.color to "blue" using inline styling.

üé® Visual Output

Before Click After Click
Black text Both paragraphs turn blue

üîÅ Alternate: Using for...of Loop

      for (let p of paragraphs) {
        p.style.color = "blue";
      }

üí° Bonus: Make Each Paragraph a Different Color

      const colors = ["red", "green", "blue"];
      for (let i = 0; i < paragraphs.length; i++) {
        paragraphs[i].style.color = colors[i % colors.length];
      }

üé® Now each paragraph cycles through colors like red, green, blue.

‚ú® Practice Exercise:

  • Try selecting all <h1> or <li> tags and apply a new background color.
  • Use console.log() to print each element's content in the loop.
  • Change font size for all paragraphs using style.fontSize.
  • Mix innerHTML and textContent to add icons or emojis to each element.

üîπ 3. getElementsByClassName()

‚úÖ Use: Find all elements with a specific class name.

      </script>

üîç What Happens Here?

  • üì¶ HTML Structure:
    • You have two <div> elements with class box ‚Üí Box 1 and Box 2.
    • You have a button labeled "Highlight".
  • üéØ When Button is Clicked:
    • highlightBoxes() function is called.
    • It uses: 
      document.getElementsByClassName("box");
      • This returns a live HTMLCollection of all elements with the class box.
  • üñåÔ∏è Then, a loop: 
          for (let i = 0; i < boxes.length; i++) {
        boxes[i].style.backgroundColor = "lightgreen";

    • Changes the background color of each box to light green.

üîπ 4. setAttribute()

‚úÖ Use: Set or update an attribute of an element (like id, class, href, src, style, etc.).

‚úÖ Example:

      <img id="myImage" src="old.jpg" width="100">
      <button onclick="updateImage()">Change Image</button>
      
      <script>

          const img = document.getElementById("myImage");
          img.setAttribute("src", "new.jpg");
          img.setAttribute("width", "200");
        }
      </script>

üîç How It Works

  • üñºÔ∏è Initial Image: 
    <img id="myImage" src="old.jpg" width="100">
    • Displays an image (old.jpg) with width of 100px.
  • üñ±Ô∏è Button: 
    <button onclick="updateImage()">Change Image</button>
    • When clicked, it triggers the updateImage() function.
  • ‚öôÔ∏è JavaScript Function: 
          const img = document.getElementById("myImage");
      img.setAttribute("src", "new.jpg");
      img.setAttribute("width", "200");
    • Gets the <img> element by its ID.
    • Updates its src attribute to new.jpg.
    • Changes the width to 200.

üñºÔ∏è Visual Result

Before Click After Click
Image: old.jpg, width: 100px Image: new.jpg, width: 200px

‚úÖ Alternative (More Common Way):

You can also change the attributes using dot notation:

      img.width = 200;

Both approaches are valid, but this method is more direct and readable ‚úÖ

üîπ 5. createElement()

‚úÖ Use: Create new HTML elements dynamically using JavaScript.

‚úÖ Example:

          newItem.textContent = "New List Item";

        }
      </script>

üîç Step-by-Step Explanation

  • üìã Initial HTML: 
          <ul id="list">
        <li>Item 1</li>
      </ul>
    • Contains one list item.
  • üñ±Ô∏è Button:
    • When clicked, it runs addItem().
  • ‚öôÔ∏è Function Execution: 
          const newItem = document.createElement("li");
      // Creates a new 
  •  element
      
      newItem.textContent = "New List Item";
      // Sets its text content
      
      document.getElementById("list").appendChild(newItem);
      // Appends it to the end of the list

üîÅ Result After Clicking "Add Item" Twice:

      <ul>
        <li>Item 1</li>
        <li>New List Item</li>
        <li>New List Item</li>

‚ú® Optional: Add Item Numbering

Want each new item to be numbered automatically? Use a counter:

      }

Now the list will grow like:

      Item 2
      Item 3
      Item 4 ...

üîπ 6. innerHTML vs textContent

‚úÖ Use: Both are used to get or set content inside an HTML element, but they behave differently.

Property Handles HTML Tags Renders HTML Treats as Plain Text
innerHTML ‚úÖ Yes ‚úÖ Yes ‚ùå No
textContent ‚ùå No ‚ùå No ‚úÖ Yes

‚úÖ Example:

      <p id="demo"></p>
      
      <script>
        const demo = document.getElementById("demo");
        demo.innerHTML = "<b>Hello Bold!</b>";  // Displays bold text
        // demo.textContent = "<b>Hello Bold!</b>";  // Displays: <b>Hello Bold!</b> as text
      </script>

üîç Difference Explained:

innerHTML parses and renders HTML tags inside the content.

textContent treats everything as plain text ‚Äî including any tags.

üß† Use Cases:

  • Use innerHTML when you want to include HTML tags.
  • Use textContent when you want to prevent HTML from being rendered (e.g., for security or raw data).

üîπ 7. querySelector() and querySelectorAll()

‚úÖ Use: Select elements using CSS-style selectors ‚Äî powerful and flexible!

Method Returns Example
querySelector() First matching element document.querySelector("p")
querySelectorAll() All matching elements document.querySelectorAll(".box")

‚úÖ Example:

      <div class="box">Box 1</div>
      <div class="box">Box 2</div>
      <button onclick="highlight()">Highlight</button>
      
      <script>
        function highlight() {
          const firstBox = document.querySelector(".box");
          firstBox.style.border = "2px solid red";
      
          const allBoxes = document.querySelectorAll(".box");
          allBoxes.forEach(box => box.style.backgroundColor = "yellow");
        }
      </script>

üîç Benefits:

  • Allows you to select by tag, id, class, or even nested elements like ul li:first-child.
  • Very useful when working with dynamic or complex HTML.

üîπ 8. Traversing the DOM

‚úÖ Use: Navigate around the DOM tree using relationships like parent/child/sibling.

‚úÖ Common Properties:

Property Description
parentNode Get the parent element
childNodes Get all child nodes (including text)
firstChild Get the first child node
lastChild Get the last child node
nextSibling Get the next node (can be text too)
previousSibling Get the previous node

‚úÖ Example:

      <ul id="myList">
        <li>Item 1</li>
        <li id="second">Item 2</li>
        <li>Item 3</li>
      </ul>
      
      <script>
        const secondItem = document.getElementById("second");
      
        console.log(secondItem.parentNode);        // <ul>
        console.log(secondItem.previousSibling);   // Might return a text node
        console.log(secondItem.nextSibling);       // Might return a text node
        console.log(secondItem.nextElementSibling); // <li>Item 3</li>
      </script>

üß† Pro Tips:

  • Use nextElementSibling and previousElementSibling to skip over text nodes.
  • Great for dynamic DOM operations when you don‚Äôt have direct access to an element.
