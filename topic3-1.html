3.1 Logical & Comparison Operators

Date: 03 April 2025 | Time: 06:31 AM

Comparison Operators

OperatorMeaningExample (x = 5, y = 10)
==Equal tox == y → false
!=Not equal tox != y → true
>Greater thanx > y → false
<Less thanx < y → true
>=Greater or equalx >= y → false
<=Less or equalx <= y → true

Logical Operators

OperatorMeaningExample (a = true, b = false)
&&ANDa && b → false
||ORa || b → true
!NOT!a → false

Example

let age = 20;
if (age >= 18 && age <= 30) {
  console.log("Eligible for the program");
} else {
  console.log("Not eligible");
}

Practice Questions

  1. console.log(5 > 3 && 10 < 20); // true
console.log(5 == "5");         // true (loose comparison)
console.log(5 === "5");        // false (strict comparison)
console.log(!(4 >= 4));        // false
          
  2. let age = 15;
if (age >= 13 && age <= 19) {
  console.log("Teenager");
} else {
  console.log("Not a Teenager");
}
// Output: Teenager
          
  3. console.log(true || false && false); // true

// Explanation:
// false && false → false
// true || false → true
